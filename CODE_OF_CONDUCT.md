# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

- Be respectful and constructive.
- Focus on ideas and behavior, not individuals.
- Assume positive intent and collaborate in good faith.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Reporting

Report unacceptable behavior privately to the maintainers via security/contact channels documented in [SECURITY.md](SECURITY.md).
